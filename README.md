# GLITCH-GARDEN
This is a farming game made in Godot. It also contains a store from where you can buy upgrades It has Days and levels a day of 20 seconds With each day and level the diffcuilty increases

Instructions: Click Water to grow, Click Light to keep it alive. To go to the next level you have to increase the light level to go above 50% and at the same time the water level should be 100%
